Quick start
-----------

Clone the repo, `git clone git://github.com/everilae/auduino.git`, then run:

```
cd auduino
make
```

Building with debugging
-----------------------

```
env DEBUG=1 make
```

Uploading to device
-------------------

```
make upload
```
